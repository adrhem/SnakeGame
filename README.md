## Snake Game

Un juego de snake hecho en C++ cuando aún cursaba la universidad. Posiblemente fue hecho en el año 2012.
Recientemente lo he actualizado para que funcione en Windows y Linux. Además cambié conio.h por [conio_am.h](https://sourceforge.net/projects/coniohcloneturboccpp/) para que compile con GCC usando un `Makefile`.
